⚙ AI Engine Settings

Choose which AI engine generates case summaries. The rest of the system (wizard, documents, tracking, printing) works fully offline with no AI needed.

Groq
FREE
Fastest free AI available. Uses Llama 3 / Mixtral models. Free tier — no credit card. You likely already have a key. Works anywhere with internet.
Free · ~2s response · Internet required
Get a free key at console.groq.com → API Keys → Create key. Paste it above. It's saved in your browser.
Ollama (Local / Offline)
OFFLINE
Runs entirely on your laptop HDD. No internet, no API key, no cost — ever. Best for court demos where WiFi is unreliable. Requires one-time setup.
Checking if Ollama is running…
One-time setup (takes ~10 minutes):
1. Download from ollama.com and install (Windows / Mac / Linux)
2. Open Terminal and run: ollama pull llama3.2
3. Then run: ollama serve
4. Come back here — it will connect automatically on localhost:11434

Model options (choose one to pull):
• ollama pull llama3.2 — 2GB, fast, good quality ✓ Recommended
• ollama pull mistral — 4GB, excellent quality
• ollama pull phi3 — 2GB, very fast on low-end laptops

No internet needed after setup. Works forever for free.
Claude (Anthropic)
FUNDED
Best quality for legal documents in Sinhala, Tamil and English. Activate when the project receives government funding. Uses claude-sonnet model.
Paid · Best legal quality · Activate when funded
Get from console.anthropic.com after funding. Paste your API key above.
Offline — No AI Summary
OFFLINE
All system features work fully. Case summaries are filled in manually by the Field Officer or skipped. Ideal for demonstration if no internet or setup is available.
Always works · No setup · No internet needed
For your Chief Justice proposal: Use Groq if the venue has WiFi (fastest, free). Use Ollama if WiFi is unreliable — set it up on your laptop the night before. Use Offline if you just want to demonstrate the workflow without AI.
